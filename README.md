# CycleGAN_Snowdrop_Removal

レンズに付着した雪の粒をCycleGANで除去するシステム

## 概要

このプロジェクトは、カメラレンズに付着した雪の粒子を自動的に除去するためのCycleGAN（Cycle-Consistent Adversarial Networks）を使用した深層学習システムです。ペアになっていない画像データセットを使用して、雪で汚染された画像をクリーンな画像に変換する画像間変換を行います。

## ファイル構成

### 主要ファイル

#### `config.py` - 設定とモデル定義
- **目的**: モデルアーキテクチャとハイパーパラメータの中央管理
- **主要設定**:
  - `cellSize = 512`: 画像の処理サイズ（512x512ピクセル）
  - `epochSize = 3`: 学習エポック数（デフォルトは100だが現在は3に設定）
  - `batchSize = 4`: バッチサイズ
  - `resBlocks`: Residualブロック数（cellSizeに応じて6または9）
  - `lambda_identity = 1`: アイデンティティ損失の重み
  - `lambda_cycle = 1`: サイクル一貫性損失の重み

- **主要クラス**:
  - `Generator`: エンコーダ・デコーダアーキテクチャの画像変換ネットワーク
  - `Discriminator`: 真偽判定を行う畳み込みネットワーク
  - `ResidualBlock`: Generatorの構成要素となる残差ブロック
  - `ImagePool`: 学習安定化のための画像バッファ
  - `loss_scheduler`: 学習率減衰スケジューラ

#### `load_dataset.py` - データセット読み込み
- **目的**: 2つのドメイン（雪あり/雪なし）の画像データセットを読み込み・前処理
- **主要クラス**:
  - `ImageFolder_2dom`: ペアになっていない2ドメインの画像データセット処理
- **データ変換**:
  - `data_transforms_crop`: 回転、反転、クロップを含む拡張処理
  - `data_transforms_only`: テンソル変換のみの基本処理
- **機能**:
  - 画像サイズに応じた適応的前処理
  - データ拡張による学習データ増強
  - PyTorch DataLoaderとの統合

#### `train.py` - 学習メインループ
- **目的**: CycleGANの学習オーケストレーション
- **学習プロセス**:
  - 2つのGenerator（G_A2B, G_B2A）と2つのDiscriminator（D_A, D_B）の交互学習
  - 敵対的損失、サイクル一貫性損失、アイデンティティ損失の組み合わせ
  - ImagePoolを使用した学習安定化
- **出力**:
  - 学習済みモデルの保存（.pthファイル）
  - 学習進捗の可視化画像
  - 損失推移のCSVログ

#### `predict_1img.py` - 単一画像推論
- **目的**: 学習済みモデルを使用した単一画像の雪除去処理
- **使用方法**: `python predict_1img.py モデルファイル名 画像ファイル名`
- **処理フロー**:
  1. 学習済みGeneratorモデルの読み込み
  2. 入力画像のリサイズと前処理
  3. モデルによる推論実行
  4. 出力画像の後処理と保存（元サイズにリサイズ）
- **出力**: `{元ファイル名}_cg.png`として保存

#### `predict_1img_outputpath.py` - 出力パス指定推論
- **目的**: 出力先を指定できる単一画像推論
- **使用方法**: `python predict_1img_outputpath.py モデルファイル名 画像ファイル名 出力パス`
- **機能**: `predict_1img.py`と同様だが、出力先ディレクトリを指定可能
- **出力**: 指定パスに`{元ファイル名}_cg.jpg`として保存

#### `requirements.txt` - 依存関係
- **主要依存関係**:
  - `torch==2.7.1`: PyTorch深層学習フレームワーク
  - NVIDIA CUDA関連ライブラリ（GPU加速用）
  - その他の必要なPythonパッケージ

## アーキテクチャ概要

### CycleGANの構成要素

1. **Generator (G_A2B, G_B2A)**:
   - エンコーダ: 畳み込み層による特徴抽出
   - Residualブロック: 深層ネットワークでの勾配消失問題を解決
   - デコーダ: 転置畳み込み層による画像再構成

2. **Discriminator (D_A, D_B)**:
   - 畳み込みネットワークによる真偽判定
   - LeakyReLU活性化関数とInstanceNormalizationを使用

3. **損失関数**:
   - 敵対的損失: GeneratorとDiscriminatorの競争学習
   - サイクル一貫性損失: A→B→Aの変換で元画像を復元
   - アイデンティティ損失: 同一ドメインでの色保持

### データフロー

```
学習時: load_dataset.py → train.py ← config.py
推論時: config.py → predict_*.py → 学習済みモデル
設定: config.py が全コンポーネントの中央設定として機能
```

## 使用方法

### 学習
```bash
python train.py 実験ID ソース画像ディレクトリ ターゲット画像ディレクトリ
```

### 推論
```bash
# 基本推論
python predict_1img.py モデルファイル.pth 入力画像.jpg

# 出力パス指定推論
python predict_1img_outputpath.py モデルファイル.pth 入力画像.jpg 出力ディレクトリ
```

## 技術的特徴

- **ペアなし学習**: 対応する雪あり/雪なし画像ペアが不要
- **リアルタイム推論**: 学習済みモデルによる高速な単一画像処理
- **レンズ特化**: 環境の降雪ではなく、カメラ機器に付着した雪粒子に焦点
- **画質保持**: 元画像の寸法と視覚的品質を維持

このシステムは、自動運転車両、監視システム、屋外モニタリングアプリケーションなど、雪の多い環境での信頼性の高い視覚認識を可能にします。
